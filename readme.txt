== Description ==

Keystone Elements Addons is a premium addon for the Elementor Page Builder plugin. It provides additional elements and features to enhance your Elementor-powered website.

== Installation ==

1. Upload the `keystone-elements-addons` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
